À propos du site

L'intégralité du site a été réalisée en Javascript et en HTML. Les aides ont été analysées et ajoutées à la main, ce qui explique qu'il peut en manquer quelques unes : si vous notez une erreur de ce type, n'hésitez pas à m'envoyer un mail !

Je souhaite aider

Vous souhaitez aider au développement de cet outil ? Tout d'abord, merci ! Que vous sachier programmer ou non, vous pouvez nous aider. Voici trois manières pour cela :

  • Vous savez programmer ? Vous pouvez corriger des bugs ou ajouter des aides proposées en récupérant le code source du site sur GitHub (voir plus bas). Ma réactivité peut varier, mais je fais de mon mieux pour évaluer et accepter les pull requests régulièrement :)
  • Vous connaissez des aides qui ne sont pas présentes ? Indiquez-le nous par mail ! Ainsi, nous pourrons les intégrer au plus vite dans la prochaine version du site !
  • Vous voulez nous aider à nous faire connaitre ? Mes aides est un site gratuit et n'a pas pour objectif de me rapporter de l'argent. En revanche, j'aimerais pouvoir aider autant de gens que possile ; donc n'hésitez pas à le partager sur vos réseaux sociaux !

Open source

Parce que ce type d'informations est trop important pour que je puisse y travailler sans contrôle et sans participation, l'intégralité du site est sous licence Open Source (licence MIT). Vous pouvez donc récupérer l'intégralité du code source
